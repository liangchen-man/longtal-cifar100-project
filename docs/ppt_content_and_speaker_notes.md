# PPT 逐页内容与讲稿

1. 封面：CIFAR-100-LT 图像长尾分类实验。
2. 背景：真实世界数据天然长尾，头部类主导训练。
3. 任务与数据集：exp/step 构建训练集，balanced 与 imbalanced 双测试集。
4. 难点：数据、梯度、边界、校准四类偏置。
5. 总体方案：HOG+SVM、ResNet-CE、LDAM-DRW、Balanced Softmax。
6. 传统基线：HOG 特征、PCA、Linear SVM。
7. 神经网络基线：ResNet-32 与 CE。
8. LDAM-DRW：类别相关 margin 与后期再加权。
9. Balanced Softmax：类先验修正。
10. 实验设置与评价指标。
11. 结果与错误分析：结果表、类别准确率、混淆矩阵。
12. 总结与展望：MiSLAS、RIDE、PaCo、VLM。
